Background
background Shorthand: allows several background properties to be defined at once MDN Info
background-attachment Sets whether a background image's position is fixed or scrolls with the containing block. MDN Info
background-color Changes the color of the background MDN Info
background-image Sets one or more background images on an element MDN Info
background-position Sets the inital postion for each background image (relative to background-origin). MDN Info
background-repeat Sets how background images are repeated (horizontal/vertical) MDN Info
Borders
border Shorthand: allows several border properties to be set at once (can also be specifically applied to border-top, -left, -right and -bottom). MDN Info
border-color Sets the color of an element's border (can also be specifically applied to border-top, -left, -right and -bottom) MDN Info
border-style Sets the line style of the element's border (can also be specifically applied to border-top, -left, -right and -bottom) MDN Info
border-width Sets the width of the element's border (can also be specifically applied to border-top, -left, -right and -bottom) MDN Info
Text & Fonts
color Sets the foreground color value of an elemnt's text and text decorations MDN Info
font Shorthand: allows several properties to be set at once MDN Info
font-family Specifies a prioritized list of one or more font family names MDN Info
font-size Sets the size of the font MDN Info
font-style Sets the style of the font (e.g., italics) MDN Info
font-variant Shorthand: allows several font variants to be set at once MDN Info
font-weight Sets the weight (or boldness) of the font MDN Info
letter-spacing Sets the horizontal spacing behavior between text characters/td> MDN Info
text-align Sets the horizontal alignment of a block element or table-cell box (left, right, center or justify) MDN Info
text-decoration Shorthand: sets the appearance of decorative lines on text MDN Info
text-indent Sets the length of empty space before lines of text in a block MDN Info
text-transform Sets how an element's text is capitalized MDN Info
Positioning & Spacing
clear Sets whether an element must be moved below (cleared) floating elements that precede it MDN Info
float Places an element on the left or right side of its container, allowing text and inline elements to wrap around it MDN Info
height Specifies the height of an element MDN Info
left With top, right and bottom, participates in specifying the horizontal position of a positioned element MDN Info
margin Sets the margin area on all four sides of an element (the side can also be specified with margin-top, -left, right and -bottom) MDN Info
padding Sets the padding area on all four sides of an element (the side can also be specified with padding-top, -left, right and -bottom) MDN Info
position Sets how an element is positioned in a document (top, left, right and bottom determine the final location of positioned elements) MDN Info
vertical-align Sets the verticle alignment of an inline, inline-block or table-cell box MDN Info
z-index Sets the z-order of a positioned element and its descendants or flex items; elements with a larger z-index cover those with a smaller one MDN Info
Other Properties
clip Defines a visible portion of an element (basically crops it); applies only to absolutely/fixed positioned elements MDN Info
cursor Sets the cursor to use when the mouse is over an element MDN Info
display Sets whether an element is treated as a block or inline element and the layout used for its children MDN Info
filter Applies graphical effects like blur or color shift to an element MDN Info
list-style Shorthand: allows several list style properties to be set at once MDN Info
list-style-image Sets an image to be used as the list item marker MDN Info
list-style-position Sets the psotion of the marker relative to a list item MDN Info
list-style-type Sets the marker (such as a disc etc.) of a list item element MDN Info
overflow Sets the desired behavior for an element's overflow; i.e., when an element's context is too big to fit in its block formatting context MDN Info
visibility Shows or hides an element without changing the layout of a document MDN Info
